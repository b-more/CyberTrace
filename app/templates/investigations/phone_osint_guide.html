Section 1: Phone OSINT Investigation Overview
What is Phone OSINT?

Phone Open Source Intelligence (OSINT) is the systematic collection and analysis of publicly available information about a phone number from multiple sources including telecommunications databases, social media platforms, public records, and search engines.

What This Investigation Provides

When you run a phone investigation in CyberTrace, the system automatically:

  1. Validates the Number - Checks if valid and properly formatted
  2. Identifies the Carrier - Determines network provider
  3. Locates Registration - Country and region
  4. Classifies Number Type - Mobile, landline, VoIP, etc.
  1. Assesses Risk - Calculates fraud risk score
  2. Generates Social Media Links - Direct verification methods
  3. Creates Google Dorks - 50+ specialized searches
  4. Checks Spam Databases - Scam reporting sites
Investigation Capabilities & Limitations
What We CAN Find (OSINT) What We CANNOT Find (Requires Legal Authority)
  • Number validity and format
  • Carrier/network provider
  • Registration country/region
  • Number type (mobile/VoIP/landline)
  • Public social media accounts
  • Publicly posted information
  • Scam/spam reports
  • Business directory listings
  • TrueCaller/crowd-sourced names
  • Subscriber name Carrier Request
  • Subscriber address Carrier Request
  • ID/Passport number Carrier Request
  • Current real-time location Court Order
  • Call history/CDRs Court Order
  • SMS message content Court Order
  • Private social media posts Court Order
  • WhatsApp/Telegram messages Court Order
Important Legal Note: This OSINT tool only gathers publicly available information. For subscriber details, call records, SMS content, or real-time location tracking, you must obtain proper legal authorization through court orders or official carrier requests following Zambia Police Service protocols and the Data Protection Act.
Section 2: Complete Investigation Workflow

Follow this systematic 6-step approach for every phone number investigation:

STEP 1: Initial Assessment (First 2 Minutes)
What to Do:
  1. Review Risk Assessment Section (top of results page)
    • Check the Risk Score (0-100 scale)
    • Read all Risk Flags carefully
    • Note HIGH RISK indicators: VoIP, Premium Rate, Foreign, Unknown Carrier
    • Review Recommendations provided by system
  2. Examine Phone Validation Section
    • Verify number is valid and properly formatted
    • Check number type (Mobile, Landline, VoIP, Premium Rate)
    • Note country code and registration location
    • Review all three number formats (International, National, E.164)
  3. Check Carrier Information Section
    • Identify the telecommunications provider
    • If "Unknown Carrier" - RED FLAG for potential VoIP/burner
    • Known carriers (Airtel, MTN, Zamtel) = easier to trace
    • Note if carrier information matches expected location
What to Document:
Information Where to Find Why It Matters
Risk Score Risk Assessment section (colored bar) Determines investigation priority and next steps
Number Type Phone Validation section VoIP = High fraud risk; Mobile = Standard; Premium = Billing scam
Carrier Name Carrier Information section Needed for subscriber information requests
Country/Location Location Information section Foreign numbers may require international cooperation
CRITICAL ACTION: Download PDF report immediately to preserve timestamped evidence before continuing investigation.

STEP 6: Determine Next Investigation Actions

Based on your findings, follow the appropriate action pathway:

IMMEDIATE ACTIONS REQUIRED:
  1. Escalate to Senior Investigator
    • Brief supervisor on high-risk indicators
    • Request priority handling
    • Consider case for specialized units
  2. Request Carrier Subscriber Information
    • Draft official letter to carrier (Airtel/MTN/Zamtel)
    • Request: Subscriber name, ID number, registration address
    • Required information: Case number, court order (if needed), investigating officer details
    • Typical response time: 3-7 business days
  3. Consider Call Detail Records (CDR) Request
    • Requires court order
    • Shows: All calls/SMS in/out, timestamps, durations, cell tower locations
    • Brief prosecutor on need for CDR warrant
    • Prepare warrant application with evidence supporting probable cause
  4. Alert Other Units if Necessary
    • Cybercrime Unit: If sophisticated technical indicators
    • Financial Crimes: If money laundering suspected
    • Organized Crime: If evidence of criminal network
    • International Cooperation: If foreign number
  5. Protective Measures
    • Warn potential future victims if pattern identified
    • Consider public advisory if widespread threat
    • Coordinate with victim support services

DEEP PROFILE ANALYSIS STEPS:
  1. Complete Profile Documentation
    • Screenshot entire profile (every section)
    • Save all photos/videos (right-click → Save)
    • Document friends/followers list
    • Copy all bio/about information
    • Note account creation date
  2. Timeline Analysis
    • Review post history chronologically
    • Identify posts relevant to investigation
    • Look for location tags/check-ins
    • Note communication patterns and associates
    • Screenshot posts that show criminal intent/activity
  3. Network Mapping
    • Document frequent contacts/commenters
    • Identify possible accomplices
    • Check if connected to other suspects/victims
    • Map relationship network
  4. Cross-Platform Correlation
    • Compare usernames across platforms
    • Check if same profile picture used elsewhere
    • Run reverse image search on profile photos
    • Look for bio mentions of other social media
  5. Preservation Request
    • Draft preservation letter to platform (Facebook/Instagram/Twitter)
    • Request: Account data preservation pending legal process
    • Prevents suspect from deleting evidence
    • Typically valid for 90 days, renewable
    • Follow up with legal process (court order) for actual data retrieval

SPECIALIZED INVESTIGATION PATH:
  1. Unknown Carrier - Domestic Number
    • Contact ZICTA (Zambia Information and Communications Technology Authority)
    • Request: Carrier identification for the number
    • ZICTA maintains master registry of number allocations
    • Once carrier identified, proceed with subscriber request
  2. Foreign Number Investigation
    • Identify country from country code (+260=Zambia, +254=Kenya, +255=Tanzania, etc.)
    • Determine if international cooperation needed
    • Options:
      • Interpol Route: Contact Interpol NCB Lusaka for international request
      • MLAT Route: Mutual Legal Assistance Treaty request through Attorney General
      • Direct Route: If neighboring SADC country, may have bilateral agreements
    • Document: Why foreign number involved in local crime (suspect fled? International network? Victim traveling?)
    • Prepare complete case brief for international request
  3. VoIP Investigation (if identified as VoIP)
    • VoIP numbers often untraceable to physical person
    • Identify VoIP provider if possible (Skype, Google Voice, etc.)
    • VoIP providers rarely cooperate without US/international court orders
    • Focus investigation on: Associated social media, payment methods, IP addresses
    • Consider this a sophisticated threat - likely tech-savvy suspect

ALTERNATIVE INVESTIGATION STRATEGIES:

Remember: No online presence does NOT mean innocent. Many legitimate people avoid social media. Many criminals use good OPSEC. Continue investigation through non-OSINT methods:

  1. Focus on Carrier Data Requests
    • Subscriber information becomes primary lead source
    • Request full registration details: Name, ID, address, registration date
    • If serious case: Request CDR (calls/SMS patterns reveal associates)
  2. Cross-Reference with Case Evidence
    • Check victim's phone: Is number saved? Under what name?
    • Review victim's SMS/call history with the number
    • Examine victim's social media: Did suspect contact them there first?
    • Bank records: Any transactions linked to this number?
  3. Witness Interviews
    • Ask victim: How did you get this number? Who gave it to you?
    • Show victim WhatsApp profile picture (if exists): Do they recognize?
    • Interview victim's associates: Anyone recognize the number?
  4. Traditional Investigation
    • Call the number (from different phone): What voicemail greeting? (Record it)
    • Check business directories: Yellow Pages, industry listings
    • Government databases: Tax records, business registrations (with proper authority)
Statistical Context: In Zambia, approximately 60-70% of mobile numbers have NO significant online footprint. This is normal. Lack of online presence is NOT an indicator of innocence or guilt - proceed with standard investigation protocols.

HIGH-PRIORITY THREAT RESPONSE:
Significance: Disposable or VoIP numbers indicate sophisticated criminal. This is NOT a casual offender. Suspect has taken steps to avoid identification.
  1. Immediate Case Escalation
    • Brief senior investigator on sophistication indicators
    • Request cybercrime unit involvement
    • Consider organized crime investigation
    • Treat as high-priority case
  2. Document Sophistication Level
    • Record exact disposable SMS site where found
    • Screenshot showing number on site
    • Note if number used for OTP/verification codes
    • This proves suspect's technical capability
  3. Pattern Analysis Across Cases
    • Search CyberTrace database: Any other cases with VoIP/disposable numbers?
    • Look for similarities: Scam tactics, amounts, victim demographics
    • May indicate same criminal organization
    • Consider linking cases
  4. Expanded Investigation Focus
    • Payment Trails: How did money move? (Mobile money, bank, crypto)
    • Associated Accounts: Email addresses, social media used in scam
    • IP Addresses: If any digital communication, trace IP
    • Network Analysis: Likely part of larger operation - identify cell
  5. Multi-Agency Coordination
    • Cybercrime Unit: Technical investigation
    • Financial Intelligence Centre: Money laundering investigation
    • Interpol: If international aspects
    • Drug Enforcement Commission: If organized crime suspected

Section 4: Risk Assessment Interpretation
Understanding the Risk Score

The Risk Assessment section provides an automated fraud threat evaluation based on multiple indicators including number type, carrier identification, location, and validation status.

Risk Score Ranges & Meanings:
Risk Score Classification Interpretation & Actions
0-39 LOW RISK Interpretation: Standard phone number with no major fraud indicators. Likely legitimate subscriber.
Actions:
  • Continue normal investigation procedures
  • Verify subscriber information if needed
  • Low priority for urgent carrier requests
40-69 MODERATE RISK Interpretation: Some concerning indicators present. Could be legitimate but warrants enhanced scrutiny.
Actions:
  • Enhanced verification recommended
  • Request carrier subscriber information
  • Thorough social media verification
  • Cross-reference with other evidence
70-100 HIGH RISK Interpretation: Multiple serious fraud indicators detected. High probability of criminal activity.
Actions:
  • Prioritize investigation immediately
  • Escalate to senior investigator
  • Request CDR with court order
  • Consider organized crime involvement
  • Coordinate with specialized units
Risk Flags Explained:

Risk flags are specific warnings that contributed to the risk score. Each flag indicates a potential fraud indicator:

What it means: Voice over Internet Protocol - phone number that operates over the internet instead of traditional mobile/landline network.

Why it's high risk:

  • Anonymous Registration: VoIP numbers can be obtained without official ID verification
  • Difficult to Trace: Not tied to physical SIM card or location
  • No Carrier Records: Traditional carriers can't provide subscriber information
  • Easily Disposable: Can be discarded and replaced instantly
  • International Access: Can appear as local number while operated from anywhere globally
  • Commonly Used by Scammers: Preferred by sophisticated fraud operations

Common VoIP Services:

  • Skype (Microsoft)
  • Google Voice
  • WhatsApp calling (uses existing number)
  • Viber Out
  • Various online SMS services

Investigation Actions:

  1. Document VoIP detection in report
  2. Escalate case priority - sophisticated threat
  3. Focus on: Associated email addresses, payment methods, IP addresses
  4. Request data from VoIP provider (often requires international cooperation)

  • Missed Call Fraud: Call and hang up, victim calls back expensive number

  4. Contact carrier to identify premium rate provider

  6. Check if multiple victims targeted with same tactic
Legal Note: Premium rate numbers are sometimes legitimate (business helplines, adult services) but frequency use in fraud schemes warrants enhanced scrutiny.
